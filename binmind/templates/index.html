Как подключить Ghidra и LLM

BinMind сам по себе — это интерфейс. Чтобы он отвечал, нужны две вещи, запущенные на твоём компьютере:
1. Запусти Ghidra (Docker)
Нужен установленный Docker. Выполни в терминале:
docker run --rm -p 9090:9090 -v binmind_data:/data/ghidra_projects biniamfd/ghidra-headless-rest:latest
2. Запусти LLM (Ollama — бесплатно)
Установи Ollama с ollama.com, затем:
ollama serve ollama pull qwen2.5-coder:7b
Или укажи облачный OpenAI-совместимый API в настройках (URL + ключ + модель).
3. Проверь подключение
Открой ⚙ Настройки, проверь адреса, нажми «Проверить связь». Индикаторы Ghidra и LLM вверху загорятся зелёным.
